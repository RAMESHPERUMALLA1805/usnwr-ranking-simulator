# 🏥 USNWR Hospital Ranking Simulator

An executive-level interactive dashboard to analyze and simulate U.S. News & World Report hospital rankings.

## 🚀 Features
- Specialty ranking analysis (Cancer – 2019–2024)
- Top 20 / Honor Roll comparison
- What-if simulation engine
- Rank improvement recommendation engine
- Interactive executive dashboard

## 🧠 Use Cases
- Identify why rankings dropped
- Compare with top hospitals
- Find highest-impact improvement areas
- Simulate rank movement before real-world changes

## ⚙️ Planned Tech Stack
- React (Vite)
- Tailwind CSS
- Recharts
- Python
- Local JSON data model

## 📊 Planned Modules
- Filter panel
- KPI cards
- Trend charts
- Opportunity table
- What-if engine
- Recommendation engine

## 🎯 Goal
Help hospital leadership answer:

**What exactly do we need to improve to reach Top 10?**
